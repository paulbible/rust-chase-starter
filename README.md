# rust-chase-starter

## Complete the following challenges.
 1) Have the program update the goal to a random position when the player touches the goal.
 2) Enforce bounds so that the player cannot move outside the screen area. 
 3) Add a scoring mechanism to the game that keeps track of the number of times that the player catches the goal.
 4) Add a win screen for when the player catches the goal a certain number of times.
